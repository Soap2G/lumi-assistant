## Notes for the model

- v3.0.1 has 36 `std:doc` pages (v3.0.4 has 37 — one added later; the parameter
  pages are identical between the two).
- The inventory also exposes section anchors (`std:label`) and the 50
  `std:cmdoption` entries; the 24 user-facing CLI options above are the ones that
  matter. For a deep-link to a sub-section, fetch the page and use its `#anchor`.
- `searchindex.js` exists but is stemmed/minified — prefer this map or a `site:`
  WebSearch over parsing it.
